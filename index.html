How it works

Simple, secure, predictable.

  1. You share files securely (usually a OneDrive/Google Drive link).
  2. I confirm scope + fixed price and a turnaround date.
  3. I deliver the output (file(s) + instructions if needed).
  4. Files are deleted after delivery unless you request otherwise.

File sending note

Email isn’t for huge attachments. If files are large, share a link (Drive/OneDrive/Dropbox/WeTransfer). Read-only access is fine.
